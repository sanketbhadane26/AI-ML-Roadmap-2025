# Image Classification using Transfer Learning (Week 5)

## Objective
Build an image classifier using a pretrained CNN on a custom dataset.

## Techniques Used
- Transfer Learning
- Data Augmentation
- Feature Extraction
- Fine-Tuning

## Model
- MobileNetV2 (ImageNet pretrained)

## Outcome
- High accuracy with limited data
- Reduced training time
- Robust image classification model
